# 🎓 COMSATS Student Portal

## 🌟 Overview
A responsive and interactive Student Portal built with HTML5, CSS3, and JavaScript. This project serves as a front-end simulation of a university portal, providing an engaging web experience with multi-page navigation, semantic structure, dynamic profile management, and smooth custom CSS animations.

@uthor Muhammad Abu_Bakr

## ✨ Key Features
- **Simulated Entry Point (`index.html`)**: An immersive loading screen featuring animated spinners, bouncing dots, and an automatic redirection progress bar.
- **Modern Landing Page (`Home.html`)**: A professionally designed home screen with a hero section, background video integration, dynamic news grid, and comprehensive campus details.
- **Student Registration Form (`Registration.html`)**: An interactive client-side form boasting real-time validation and dynamic password match confirmation.
- **Dynamic Profile Dashboard (`Profile.html`)**: Retrieves registered student information dynamically from browser `LocalStorage` mapping the details into a visually neat structure. Includes secure logout capability.
- **Real-Time Clock Engine**: Features a live Date & Time display embedded in the portal footer, updating precisely every second.

## 🛠️ Technologies Used
- **HTML5**: Meaningful, semantic page structure, and comprehensive multi-page routing layout.
- **CSS3**: Responsive flexbox/grid layouts, elegant glassmorphism effects, variables, and keyframe-based loading animations.
- **JavaScript (ES6+)**: DOM manipulation, interactive form validation, interval/timeout handlers, and robust state management via the `LocalStorage` API.

## 🚀 Getting Started
1. **Clone the Repository**: Download or clone the project folder to your local machine.
2. **Launch the Portal**: Open `index.html` in your preferred modern web browser (Google Chrome, Firefox, or MS Edge).
3. **Experience the Flow**: Watch the entry animations and let the auto-redirect guide you to the `Home` page.
4. **Register a Profile**: Navigate to **Registration** to create a mock student data entry, securely saved to local storage.
5. **View Dashboard**: Move over to the **Profile** section to browse your newly populated student dashboard.
